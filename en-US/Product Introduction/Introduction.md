YashanDB Cloud Manager (hereinafter referred to as YCM) is a monitoring and operation maintenance platform for YashanDB, designed for enterprise-level databases and providing a one-stop operation and monitoring management solution. This platform offers full lifecycle management for YashanDB and related resources. It provides  management, and operation of YashanDB, as well as monitoring and alert functions such as early warning and health checks. It also offers system management functions including user creation, privilege granting, and resource isolation.

Key usage scenarios: Include instance management, resource, performance, capacity, configuration management services, daily monitoring inspections, intelligent fault analysis services, as well as services related to backup recovery, high availability for stability and security. By transforming passive operations into proactive operations, it effectively enhances the database operation and maintenance level.


## Key Features


### Database Instance Management

It enables automated and visual database operational actions, reducing the routine redundant work of database administrators.

Supports one-click operations for starting, stopping, and restarting databases; visualizing database cluster topology and diversifying cluster configuration management.

Provides tablespace management, supporting the creation, deletion, and inspection of tablespace usage, as well as database privilege management, management of database auditing policies, display of audit logs, and management of session connection counts, reducing repetitive and inefficient daily tasks for database administrators.

### Resource Monitoring

It offers a comprehensive and robust monitoring metric library covering most key metrics of databases, systems, and hosts, helping users detect database issues in real time and guiding quick resolutions, thereby avoiding issues that could impact business operations.

Customizable monitoring dashboards meet users' needs to select monitoring metrics and bind specific hosts and database resources, achieving visual resource monitoring.

### Alarm Management

Provides two types of alarm strategies: general built-in alarm strategies and user-customizable alarm strategies based on the monitoring metric library, better meeting business needs for flexible anomaly detection of specific resources.

For anomalous alarm information, it offers alarm notification methods via email and SMS, allowing users to promptly receive database alarm information and ensuring database security.

### Inspection Management

Configurable inspection strategies allow for on-demand scheduled inspections of databases to diagnose existing problems, replacing manual daily inspections and reducing the workload of operations personnel.

Health inspections evaluate database health across five modules: security, stability, availability, performance analysis, and capacity analysis, summarizing existing risks and generating detailed inspection reports. Inspection information and reports are promptly pushed via email to DBAs and operational personnel.

### Backup and Recovery

It provides operational capabilities for data backup and recovery of databases to ensure high availability and security. The platform enables quick backup and verification of database data based on backup and recovery strategies, guaranteeing a complete and reliable backup recovery data set. It supports both full and incremental backups of databases, as well as one-click recovery operations, enhancing the overall system security.

### High Availability and Fault Handling Visualization

It offers six comprehensive reliability switching schemes: electable mode with maximize protection, electable mode with maximize availability, electable mode with maximize performance, non-electable mode with maximize performance, non-electable mode with maximize availability, and non-electable mode with maximize protection.

It provides primary/standby switching (switchover, failover) and related operational management capabilities for high availability, ensuring business disaster recovery capabilities and continuity.

### Efficient Database Diagnosis and Optimization

It provides comprehensive diagnostic analysis capabilities, enabling database operations personnel to efficiently perform diagnostic optimization through a visual interface.

Users can utilize functionality features such as slow SQL analysis, performance reports, snapshot management, lock analysis, and transaction analysis to conduct diagnostic analysis from multiple dimensions like SQL, database sessions, deadlocks, and capacity, thus ensuring database optimization and normal operation.

### Privilege Management

It offers complete user privilege management capabilities, allowing for user creation, classification, and granting operations through the three concepts of users, user groups, and roles. This implements most of the user management functionality required by common enterprise-level database management software.

It supports obtaining management privileges over databases through roles and utilizes user groups for resource isolation.
