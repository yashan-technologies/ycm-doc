The personal center includes four major modules: Message Center, Resource Subscribing, Personal Information, and Resource Password Management.

- [Message Center](Message Center): Displays information about changes to user subscription resources, including alarm messages, resource changes, and task messages. Historical messages will be automatically deleted if they exceed the retention period, with a default retention period of 30 days.

- [Resource Subscribing](Resource Subscribing): The platform supports subscription to database and server resources. Related alarm messages, resource changes, and task messages will be sent to users who subscribe to these resources in the form of internal messages.

- [Personal Information](Personal Information): Displays personal information such as username and role, and supports modification of login password, phone number, and email.

- [Resource Password Saved](Resource Password Saved): The platform supports saving database usernames and passwords and supports automatic login to the database based on the saved passwords. If automatic login is configured, the process will be smoother for operations that require logging into the database; otherwise, the system will require manual login to the database before proceeding with subsequent operations.

- [API Secret Access Key Management](API Secret Access Key Management): The platform supports users in creating and managing user key information, with a limit of 2 keys per user. When creating a key, a unique ak and sk will be generated. When users need to access the platform API, they can authenticate using ak and sk.
