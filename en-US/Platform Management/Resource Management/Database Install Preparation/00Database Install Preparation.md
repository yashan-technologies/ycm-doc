Please proceed to the corresponding documents in the order listed above and follow the guidelines provided to prepare for the product installation according to the examples. The [Server Preparation](Server Preparation) document only outlines YashanDB's requirements and configuration recommendations for installation resources, without specific operational examples. It should be prepared by the enterprise's backend management personnel. If consultation is needed regarding the information listed in the document, please contact our technical support.

After the server resource preparation is complete, this chapter will provide example guidance for all pre-installation preparation steps. The listed IPs, users, folder directories, and other information serve to provide a prerequisite environment for the subsequent product installation. Users should adjust these based on their actual environment.

Additionally, all initial parameters, environment settings, and user operations on the server must be performed with root privileges. If there is a privilege division in the operating system within the enterprise, all the listed steps except for software package download should be carried out by the system administrator. Upon completion, the information regarding the installation user, storage directories, and other details should be handed over to the database installation user, who will perform the subsequent operations.
