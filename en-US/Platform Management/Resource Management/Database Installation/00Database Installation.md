YashanDB provides various deployment modes. The installation process of different modes, especially the initial installation parameter configuration, varies significantly. Please choose the appropriate operation guide based on actual needs.

- [Standalone (Primary-Standby) Deployment](Standalone (Primary-Standby) Deployment)

    The installation guide for Standalone (Primary-Standby) deployment is used to set up a single-instance single-database database management system suitable for centralized transaction business.

- [YAC Deployment](YAC Deployment)

    The YAC Deployment installation guide is used to set up a multi-instance single-database database management system suitable for shared cluster transaction business.

- [Distributed Deployment](Distributed Deployment)

    The Distributed Deployment installation guide is used to set up a distributed database management system suitable for distributed analytical business.

Before proceeding with the installation operations, please ensure that you have followed all the contents listed in the [Installation Preparation](../Database Install Preparation/00Database Install Preparation) document in order. Otherwise, this may lead to installation deployment failures or result in an incorrect/unstable environment, which could severely impact the business operating in this environment.
