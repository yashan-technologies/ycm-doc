**Web Path**: **[ YashanDB ]** > **[ Installation Package ]**

**Functionality Introduction**

The management platform supports adding YashanDB installation packages through the following two methods:

- Local Upload: Multiple installation packages can be selected and uploaded simultaneously.
- Server Path: The server path where the installation packages are located on the management platform, supporting multiple installation packages under one path. Click **[ Check ]** to display all installation packages under that path.

> **Note**:
>
> If the deployment form of the management platform is high availability, the management platform will upload installation packages to each management platform instance.  
>
> If the server hosting the management platform instance has not been added to the management platform, it is recommended to manually add it to the management platform.  
>
> Please do not modify the name of the YashanDB installation package.
