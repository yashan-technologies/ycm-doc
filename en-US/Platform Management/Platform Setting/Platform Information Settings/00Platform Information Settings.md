Platform information settings include login security settings,  notification service settings, and system contacts.

- [Platform Secure Login Configuration](Platform Secure Login Configuration): Supports configuration of user login failure retry count, locking mechanism, and authentication methods.


- [Notification Service Setting](Notification Service Setting): Manages the platform's ability to push alarm information, inspection notifications, etc., in real-time to system contacts via email or SMS, increasing response speed and improving operation and maintenance efficiency. To use this functionality, the corresponding notification service must be configured.

- [System Contacts](System Contacts): By creating contacts and specifying message reception methods (email and mobile number), users can receive real-time alerts and inspection notifications from the management platform, provided that the notification service settings have been completed.
