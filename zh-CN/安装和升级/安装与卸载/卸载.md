如果想卸载管理平台，可以通过以下步骤进行清理：

1. 移除托管的数据库。
   
    - 服务端移除：使用[yasadm](../../参考指南/工具命令介绍/yasadm.html#remove)的`yasadm yasdb remove`命令将数据库解除托管。
    
    - 网页端移除：参考[移除数据库](../../平台管理/资源管理/数据库管理.html#remove)，将数据库从管理平台移除。
    
2. 移除服务器：参考[移除服务器](../../平台管理/资源管理/服务器管理.html#remove)，将服务器从管理平台移除。

3. 进入安装路径，执行卸载命令。

    ```bash
    $ cd /opt/ycm/ycm
    $ sudo ./scripts/clean.sh 
    # 根据提示，确认是否卸载，确需卸载输入yes
    ```
> **Note**：
>
> 如果在主备部署模式下，卸载时该节点的后端元数据也会被删除。如果卸载过程中删除失败的话，请在其它节点使用[yasadm](../../参考指南/工具命令介绍/yasadm.html#remove)的`yasadm instance delete`命令手动移除。

4. 清理残余文件。
   ```bash
   $ sudo rm -rf /opt/ycm
   ```
