管理平台默认只能添加管理同CPU架构的主机，如管理平台部署在x86_64上，默认只能添加x86_64的主机。若期望同时管理aarch64的主机，需要进行额外操作。

以管理平台部署在x86_64的主机，添加aarch64的主机为例（反之亦然）：

1. 准备好x86_64和aarch64的两个管理平台安装包。

2. 按照上述安装部署步骤，使用x86_64的安装包完成在x86服务器上的管理平台部署，安装目录为`/opt/ycm`。

3. 执行添加aarch64安装包命令。

    ```bash
    cd /opt/ycm/ycm
    bash ./scripts/plugin.sh add <管理平台 aarch64安装包路径>
    ```
4. 按照[服务器管理](../资源管理/服务器管理)操作，即可添加aarch64主机。 

> **Note**：
>
> 如果管理平台是高可用部署，需要在每个节点上的服务器上面执行添加aarch64安装包命令。
